# Word文档格式示例

## 支持的题型

系统支持**三种题型**，可以自由组合使用：
1. **选择题**（单选/多选）
2. **判断题**（对/错）
3. **解答题**（主观题）

## 题型格式说明

### 1. 单选题格式

```
1. 栈的特点是什么？
A. 先进先出
B. 后进先出
C. 随机存取
D. 顺序存取
```

### 2. 多选题格式

在题目中包含"多选"关键词：

```
2. 以下哪些是线性数据结构？（多选）
A. 栈
B. 队列
C. 树
D. 图
E. 链表
```

### 3. 判断题格式

**方式一：题目末尾加（）**
```
3. 栈是后进先出的数据结构。（）
```

**方式二：明确写出对错选项**
```
4. 队列是先进先出的数据结构。
对
错
```

**方式三：使用A、B选项**
```
5. 二叉树每个节点最多有两个子节点。
A. 对
B. 错
```

### 4. 解答题格式

没有选项的开放性问题：

```
6. 请简述栈和队列的区别。

7. 什么是二叉搜索树？请说明其主要特点。
```

## 注意事项

1. **问题编号**: 使用阿拉伯数字 + 句号或顿号（如：1. 或 1、）
2. **选项标识**: 
   - 选择题：使用大写字母 + 句号或顿号（如：A. 或 A、）
   - 判断题：可以直接写"对/错"，或使用"A. 对 B. 错"格式
3. **空行分隔**: 建议每个问题之间空一行，便于阅读
4. **题型识别**:
   - **多选题**: 在题目中包含"多选"、"多项"、"不定项"等关键词
   - **判断题**: 题目末尾有（），或只有两个"对/错"类选项
   - **解答题**: 没有选项的开放性问题
   - **单选题**: 有3个或以上选项的问题
5. **编码格式**: 保存为 .docx 格式（Word 2007及以上）

## 题型混合示例

Word文档中可以自由组合三种题型，例如：

**示例1：只有单选题**
```
1. 题目1？
A. 选项1
B. 选项2

2. 题目2？
A. 选项1
B. 选项2
```

**示例2：单选+判断**
```
1. 选择题
A. 选项1
B. 选项2

2. 判断题（）
```

**示例3：三种都有**
```
1. 单选题
A. 选项1
B. 选项2

2. 多选题（多选）
A. 选项1
B. 选项2
C. 选项3

3. 判断题（）

4. 解答题？
```

## 常见问题

### Q: 支持哪些题型？
A: 支持**三种题型**：
   - 选择题（单选/多选）
   - 判断题（对/错）
   - 解答题（主观题）
   
可以只使用其中一种，也可以任意组合使用。

### Q: 如何区分单选和多选？
A: 在题目中包含"多选"、"多项"、"不定项"等关键词即可识别为多选题。

### Q: 判断题如何识别？
A: 三种方式：
   1. 题目末尾加（）
   2. 直接写"对"和"错"作为选项
   3. 使用"A. 对 B. 错"格式

### Q: 解答题如何识别？
A: 没有选项的开放性问题会自动识别为解答题。

### Q: 支持其他编号方式吗？
A: 目前支持 "1." 和 "1、" 两种格式，其他格式可能无法识别。

### Q: 选项可以不按顺序吗？
A: 建议按 A、B、C、D 的顺序编写，系统会按顺序读取。

### Q: 图片和表格会被识别吗？
A: 目前仅支持纯文本，图片和表格会被忽略。

### Q: 中英文混合可以吗？
A: 可以，系统支持中英文混合内容。

### Q: 一个文档必须包含三种题型吗？
A: 不需要！可以只包含一种题型，或者任意组合。系统会自动识别每个题目的类型。
